About Our Project

This project aims to create a comprehensive graphical user interface (GUI) for administering cognitive neuroscience tests. The system is being developed in MATLAB to ensure precise timing control and seamless integration with EEG recording equipment.

Project Goals

  • Create standardized cognitive assessment tools for research settings
  • Develop intuitive GUI for both administrators and participants
  • Provide clear instructions and progress tracking
  • Support multiple test types within a unified dashboard
  • Facilitate research on sensory processing and brain development

Technical Specifications

  • Platform: MATLAB
  • Tests: N-Back (complete with 15 stimuli), Go/No-Go (in progress)
  • Features: Three difficulty levels, instruction screens, progress tracking (Image X/15)
