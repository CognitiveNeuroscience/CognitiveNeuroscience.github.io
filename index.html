Cognitive Neuroscience GUI

Interactive cognitive task experiments designed for neuroscience research with EEG integration

Welcome

Our team is developing a comprehensive GUI system for administering cognitive neuroscience tests. These tests are designed to study sensory processing and brain development in patient populations during EEG monitoring sessions.

The interface provides a clean, intuitive way for researchers to conduct standardized cognitive assessments. Our GUI handles the visual presentation, instructions, and user interaction, ready to be integrated with EEG equipment.
