# Cognitive Neuroscience GUI Project Website

A  project website for the Cognitive Neuroscience GUI project, hosted on GitHub Pages.

## Live Website

This site is live at [cognitiveneuroscience.github.io](https://cognitiveneuroscience.github.io/)

## Pages

- **Home** (`index.html`) - Main landing page introducing the project
- **About Us** (`about.html`) - Team information
- **Projects** (`projects.html`) - Overview of project

## Deploying to GitHub Pages

### Deploy from Main Branch

1. Merge this PR to the `main` branch
2. Go to your repository settings: `Settings` â†’ `Pages`
3. Under "Source", select:
   - Branch: `main`
   - Folder: `/ (root)`
4. Click "Save"
5. GitHub will automatically build and deploy your site
6. Your site will be live at [https://cognitiveneuroscience.github.io/](https://cognitiveneuroscience.github.io/)

## Local Development

To test the website locally:

```bash
# Using Python 3
python3 -m http.server 8000

# Then visit http://localhost:8000 in your browser
```

Or use any other local web server of your choice.
