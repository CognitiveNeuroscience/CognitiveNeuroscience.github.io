Our Projects

Our GUI system guides users through cognitive neuroscience tests with an intuitive workflow. Below is the complete user journey from start to finish, along with details about each cognitive task.

User Flow

The interface follows a simple, three-step process:

Step 1: Homepage - Get Started

When users first launch the application, they see a clean welcome screen with the "Cognitive Neuroscience" title and a prominent "Get Started" button.

Step 2: Task Selection

After clicking "Get Started," users choose between two cognitive tasks: N-Back (working memory) or Go/No-Go (response inhibition).

Step 3: Test Administration

Once a task is selected, users proceed to the specific test interface with instructions and the testing environment.

N-Back Test (Complete)

A working memory experiment using a series of visual stimuli (flashcards). Participants indicate when the current stimulus matches the one shown N items back in the sequence.

How it works:

Users first see an instruction screen explaining the task and allowing them to choose difficulty (1-back, 2-back, or 3-back). Then, 15 flashcards with different object images are presented one at a time. Participants press the "Match" button when they recognize that the current image matches the one from N steps ago.

Implementation Details

  • Difficulty Levels: Three levels (1-back, 2-back, 3-back) with level buttons on instructions screen
  • Stimulus Set: 15 unique flashcard images featuring everyday objects
  • Test Length: Full session of 15 trials per run
  • Progress Tracking: Real-time display showing "Image X/15"
  • User Interface: Clean card-based design with large, clear "Match" button

✅ Fully Functional

Go/No-Go Test (Complete)

A response inhibition task inspired by Simon Says, where participants type commands only when preceded by "Simon says" and withhold response when preceded by "Coach Says."

How it works:

Users first see an instruction screen explaining the task rules. Then, text commands are presented one at a time. Participants must type the command if it starts with "Simon says" but must not type anything if it starts with "Coach says." This tests impulse control and the ability to inhibit automatic responses.

Implementation Details

  • Task Format: Simon Says style game with "Simon says" (Go) and "Coach says" (No-Go) cues
  • Response Method: Text input for Go trials, no response for No-Go trials
  • Cognitive Measure: Tests response inhibition and impulse control
  • User Interface: Clear command display with text input field and progress tracking
  • Instructions: Comprehensive instruction screen explaining game rules before test begins

✅ Fully Functional

Technical Notes

Both tests are implemented in MATLAB to ensure integrations with EEG equipment.
